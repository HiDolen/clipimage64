# ClipImage64

Paste clipboard images in base64 into your markdown file.

## Usage

1. Take a screenshot
2. Use default keyboard binding `Ctrl+Alt+V` (`Cmd+Alt+V` on Mac)
3. The image will be inserted into your markdown in base64 format.
